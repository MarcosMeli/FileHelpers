Progress Notification

From the version 1.4.2 you can get notified about the progress of the operations of the engines and DataLinks.

The already implemented feature can be used in all cases, for example when you wirte a file the engine knows the total records to be written and the actual record then can be accurate in the notification. However if you read a file the engine only know the current record and not the total then informs -1.

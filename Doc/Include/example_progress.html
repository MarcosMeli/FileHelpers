Progress Notification

You can get notified about the progress of the operations of the engines and DataLinks.

This can be used in all cases. For example, when you write a file, the engine knows the total records to be written and the record currently being processed. This information is used to provide an accurate notification. However, if you read a file and the engine only knows the current record and not the total record then the total count will be -1.

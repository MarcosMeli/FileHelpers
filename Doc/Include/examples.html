Examples Section

Here you find a index of examples sorted from the easiest to the most advanced use of the library.

Easy Examples

  • Read and write all records: the more simple example of the library.
  • Read and write all records Generics: the more simple example of the library usign Generics.

  • Using the Differents ErrorModes: the differents ways to handle the errors when read and write records.

  • Master-Detail custom selector:  this is one simple example in Master-Detail processing.
  • Master-Detail common selector:  this is other simple example in Master-Detail processing.
  • Use the automatic enum converter: this example show how to define a enum and use it in the RecordClasss.
  • CommonEngine: in this example you can see how to use the static methods in this class to handle files and string without instanciate any engine.
  • Progress Notification: shows how to get notified in the progress of the operations.

Advanced Examples

  • Asynchronous Processing: in this example you learn how to use the library in an Asynchronous Mode.
  • Decimal Custom Converter this example show how to read decimal numbers that don´t have the decimal separator.
  • Name Custom Field and Converter this example show how to use a custom converter for a custom field type.
  • File Transform Engine this example show how to change the record format of a a file

DataLink Examples

  • Access DataLink example: this example show how to import/export data to/from an Access or SQL Server Database.
  • Excel DataLink example: this example show how to import/export data to/from an Microsoft Excel File.
  • CodeSmith example: this example show how to use the Code Smith Templates to automaticly generate the code.

Examples section

Here is the index of examples sorted from the easiest to the most advanced usage of the library.

Click HERE To download two example projects (C# and VB.NET)

Easy examples

  • Read and write all records: the more simple examples of the library.
  • Read and write all records Generics: the more simple examples of the library using Generics.
  • Using the different ErrorModes: the different ways to handle the errors when reading and writint records.
  • DelimitedFileEngine: Change the record options at runtime.
  • FixedFileEngine: Change the record options at runtime.
  • .NET Nullable Types: shows how the library works with this cool feature.
  • Using Events: shows how to hook to events in the engines.
  • Conditional Records: shows how to conditionally include or exclude certain records.
  • FileDiffEngine: an example that shows the features of this cool engine.
  • Master-Detail custom selector:  this is one simple example in Master-Detail processing.
  • Master-Detail common selector:  this is other simple example in Master-Detail processing.
  • Use the automatic enum converter: this example show how to define a enum and use it in the RecordClasss.
  • CommonEngine: in this example you can see how to use the static methods in this class to handle files and string without instantiating any engine.
  • Progress Notification : shows how to get notified of the progress of an operation.

Advanced examples

  • RunTime Records: Learn how to use the most flexible and advanced features of the library.
  • Asynchronous Processing: in this example you learn how to use the library in asynchronous mode.
  • Decimal Custom Converter this example show how to read decimal numbers that don't have the decimal separator.
  • MultiRecord Engine: Learn how to use the library with multiple record types.
  • Name Custom Field and Converter this example show how to use a custom converter for a custom field type.
  • File Transform Engine this example show how to change the record format of a file.

DataLink examples

  • Access DataLink example: this example show how to import/export data to/from an Access or SQL Server database.
  • Excel DataLink example : this example show how to import/export data to/from an Microsoft Excel file.
  • SqlServer Insert Records : shows how to read a file an insert its' contents to an SqlServer table
  • SqlServer Extract Records : shows how to read a SqlServer table an write the rows to a file

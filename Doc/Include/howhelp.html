How to Help to the library

Development Help

Work for the library is more and more time consuming to me and I don't have full time for it.

So if you want to help me in the development please contact me at marcos@filehelpers.com and we will find some library part that you can maintain or change

You can help in any area that you would like:

  • Development
  • MONO port
  • Automatization trought NAnt
  • Help Translation (almost to english :) because I know that is full of errors)
  • Web Site Changes
  • Forums Admin
  • And if you want, you are welcome to donate something for the library
You can change any part of the library and later you can send me your code using this excellent article as guideline:

Scott Hanselman - Example: How to contribute a patch to an Open Source Project

Another place where you can found the pending things is in the forums, post there too your ideas and general feedback

Spread to the world !!

You can vote for the article in CodeProject so that more users can know about the library and send additional feedbacks and ideas =)

You can also post in your blog or page a link to the FileHelpers home.

Motivation

It is allways good to hear what you think about the library; in the open source world we live of the user feedback and happyness =) so send any comment to me or post them in the forums, your are always welcome.
